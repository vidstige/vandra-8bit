# Vandra 8-bit
Retro, Lo-Fi, 8-bit styled JS app displaying some text and 
3d wireframe model.

## Getting started
1. Just clone repository and 
2. `npm start` (or `yarn`)

## Font 
The font is called 04b03 and is released as FREEWARE by Yuji Oshimoto (Copyright 1998-2003).
